#  Swift & Decisive: Personal CV Webpage

A high-performance, responsive CV webpage inspired by the "Lightning Returns" aesthetic. Built as part of the Software Engineering curriculum at CAMT, Chiang Mai University.

##  Live Demo
https://a2-hrmstrailblazer.github.io/my-cv/

##  Tech Stack
* **HTML5 & CSS3**: Custom grid layouts and pseudo-element animations.
* **Bootstrap 5.3.8**: Responsive framework and UI components.
* **jQuery**: Handling interactive DOM events and animations.
* **Google Fonts**: Integrated "Segoe UI" and "Roboto" for a clean, technical look.

##  Key Features & Requirements
This project fulfills the following technical requirements:
1.  **Responsive Design**: Implements Bootstrap Grid for seamless Desktop, Tablet, and Mobile viewing.
2.  **Custom Theme**: A unique "Crystal & Neon" UI inspired by Final Fantasy UI/UX.
3.  **Interactive Elements**: 
    * **Contact Reveal**: Slide-toggle decryption for email info.
    * **Dynamic Navbar**: Scroll-based transparency and shadow logic.
    * **Hover Effects**: Real-time glow synchronization on titles and skill tags.
    * **Tactile Feedback**: Scaling effects on profile imagery.
4.  **Content Sections**: 8 dedicated sections (Intro, Experience, Education, Awards, Skills, Hobby, Portfolio, and Contact).
